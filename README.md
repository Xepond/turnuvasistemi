# CyberArena Tournament System

Vite tabanlı React uygulaması.

## Local development

1. Node.js 18+ kurulu olmalı.
2. Bağımlılıkları yükle:

```bash
npm install
```

3. Geliştirme sunucusunu başlat:

```bash
npm run dev
```

4. Üretim derlemesi al:

```bash
npm run build
```

5. Yerelde üretim önizlemesi aç:

```bash
npm run preview
```

## Deploy

En kolay yol Vercel veya Netlify kullanmaktır.

- Build command: `npm run build`
- Output directory: `dist`

Hash tabanlı sayfa geçişi kullanıldığı için ek router yapılandırması gerekmez.
